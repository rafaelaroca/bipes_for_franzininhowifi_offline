Bem vindo ao Franziniho Wifi

Bem vindo ao Franzininho Wifi! Esta placa Maker Franzininho Wifi possui diversas formas de programação. Se você acessou este arquivo, é porque ela está carregada com CircuitPython.

Dessa forma, para programar a placa, você tem várias opções simples e rápidas, sem precisar instalar nenhum software, nem fazer nenhuma configuração especial. Você pode começar agora a programar o Franzininho Wifi em Python e/ou blocos a partir de computadores MAC, Windows, Linux, Raspberry Pi, a partir de um tablet ou smartphone ou até mesmo de um console de vídeo game!

Método 1

Se você tem acesso à Internet, basta clicar no link abaixo e pronto, você já terá acesso à programação por blocos da Franzininho usando a ferramenta BIPES (www.bipes.net.br):

bipes.net.br/fw


Método 2

Se você não tem acesso à Internet neste momento, você também pode usar o BIPES com o Franzininho Wifi facilmente! Basta descompactar o arquivo ZIP que esta nesta pasta, e clicar em index.html!

bipes_fran3.zip

Welcome to Franziniho Wifi

 Welcome to Franzininho Wifi! This Maker board has several proramming possibilities. If you accessed this file, it is because you have a board on your hands, and it is preloaded with CircuitPython.

In that way, to program the board, you have several quick and simple possibilities, without the need to install any software, and without the need of any special configuration. You can start programming Franzininho Wifi using Python and/or blocks from MAC, Windows, Linux and Rasperry Pi computers, or even from smartphones, tablets or game consoles!

Method 1

If you have Internet access, just click on the link below and you can start programming with blocks or Python using BIPES (www.bipes.net.br).

bipes.net.br/fw


Method 2

If you do not have Internet access right now, you can also use BIPES offline with Franzininho Wifi easily! Simply uncompress the ZIP file that is on this folder and click on index.html. Done!

bipes_fran3.zip

Enjoy Franzininho Wifi!
